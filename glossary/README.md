# Glossary

Auto-generated concept indexes for both courses.

- `GLOSSARY.md` -- combined across both courses
- `GLOSSARY-react.md` -- React Snooker course only
- `GLOSSARY-ai.md` -- AI Interview course only

Every bolded `**Term**` (1-5 words, capitalized, not a stopword) is extracted with the surrounding sentence and a citation back to the source lesson.
