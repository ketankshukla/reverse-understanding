# EPUB Courses

Auto-generated EPUB3 exports for e-readers (Kindle, Kobo, Apple Books, Boox).

## Files

- `react-snooker-course.epub`
- `ai-interview-course.epub`

## How to read

**Kindle:** email the `.epub` to your Send-to-Kindle address, or open with the free Kindle Previewer.

**Apple Books / Kobo:** double-click the file.

**Calibre:** add to library; convert if your device prefers mobi/azw3.

## Regenerate

```pwsh
python tools\convert_extra_formats.py --formats epub --clean
```
